## Navigation Guide

### Public Pages (No Login Required)

#### Home Page (`/`)
- Landing page with information about Ella Rises
- Features a hero section, image slideshow, and mission statement
- Accessible to all visitors

#### About Page (`/about`)
- Information about the organization
- Photo gallery showcasing community activities
- Accessible to all visitors

#### Login Page (`/login`)
- Login form for registered users
- Use the credentials above to access the dashboard

---

### Authenticated Pages (Login Required)

Once logged in, you'll see a "Dashboard" dropdown menu in the navigation bar. The available pages depend on your user role (Common User vs Admin/Manager).

#### Dashboard Overview (`/dashboard`)
- **Access:** All logged-in users
- Displays key metrics and analytics
- Includes an embedded Tableau dashboard for data visualization
- Shows statistics for participants, events, surveys, milestones, donations, and registrations

#### Events (`/events`)
- **Access:** All logged-in users
- View all events in the system
- Search events by name
- View event details
- **Admin/Manager Only:** Create new events, edit existing events

#### Milestones (`/milestones`)
- **Access:** All logged-in users
- View participant milestones
- Search by participant name
- Filter by milestone type
- View milestone details

#### Profile (`/profile`)
- **Access:** All logged-in users
- View and edit your user profile information
- Change password

---

### Admin/Manager Only Pages

These pages are only accessible to users with admin or manager roles.

#### Participants (`/participants`)
- View all participants in the system
- Search participants by name
- View participant details including:
  - Personal information
  - Associated milestones
  - Event registrations
- Edit participant information
- Add new participants

#### Surveys (`/surveys`)
- View Post-Event Survey responses
- Search surveys by participant name or event name
- Filter surveys by satisfaction score (1-5)
- View detailed survey responses

#### Donations (`/donations`)
- View all donations
- Search donations by participant name
- Filter donations by amount range:
  - Under $25
  - $25-$50
  - $50-$100
  - $100-$250
  - $250-$500
  - $500-$1000
  - Over $1000
- View donation details

#### Users (`/users`)
- View all registered users
- Search users by username or email
- Filter users by role (user, manager, admin)
- Add new users
- Edit user information and roles

## Features

### Search Functionality
- Most pages include a search bar
- Search automatically trims whitespace
- Search works across multiple relevant fields

### Mobile Responsive
- The website is fully responsive and works on mobile devices
- Touch-enabled interactions for mobile users
- Mobile-friendly navigation menu

### Data Management
- Create, view, edit, and search records across all data types
- Pagination for large datasets
- Filtering options on many pages

---

## Getting Started

1. Navigate to the website: https://vanillaintex.is404.net/
2. Click "Login" in the navigation bar
3. Enter your credentials (see above)
4. Once logged in, use the "Dashboard" dropdown to navigate to different sections
5. Use the search bars and filters to find specific records
6. Click on individual records to view details
7. Use the "Edit" buttons to modify records (if you have permission)
